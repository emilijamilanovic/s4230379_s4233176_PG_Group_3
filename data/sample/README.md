# Representative Data Sample

This directory contains a small anonymised sample for assignment submission. It is
well below the 10 MB limit and demonstrates both the text/NLP structure and the
network edge structure without exposing raw public usernames.

| File | Purpose |
|---|---|
| `comments_nlp_community_sample.csv` | 500 sampled comments, balanced by platform, with anonymised authors, cleaned text, sentiment, topic, and community fields. |
| `reddit_reply_edges_sample.csv` | 500 sampled Reddit directed reply edges. |
| `youtube_reply_edges_sample.csv` | 500 sampled YouTube directed reply edges. |
| `sample_manifest.csv` | Row counts and purpose for each sample file. |

The full analysis uses `data/processed/`; this directory is only a compact
representative sample for external review.
